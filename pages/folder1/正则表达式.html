# 正则表达式

regular expression :RegExp

用来处理字符串的规则。

  • 只能处理字符串
  • 他是一个规则

# 编写正则表达式

创建方式有两种

//1、字面量创建方式（两个正斜杆之间包起来的，都是用来描述规则的元字符）
let reg1 = /\d+/;

//2、构造函数模式创建，两个参数，1.元字符字符串（可以用变量拼接字符串），2.修饰符字符串
let reg2 = new RegExp("\\d+");//完全同上，第一个"\"表示转义字符，因为字符串中的“\”不是‘\’本身(也有特殊含义)，需要转义
//eg:
console.log('\d');//d
console.log('\\d');// \d

正则表达式由两部分组成：

  • 元字符
  • 修饰符

以下基于w3school文档 (opens new window)补充说明：

// 特殊元字符
\w 数字、字母、下划线中的任意一个字符
[+-]===(+|-)

/*
*(?=pattern)正向预查
*要匹配的字符串，必须满足pattern这个条件
***************
*/
let reg = /cainiao(?=8)/;
let str1 = 'cainiao9';
alert(reg.exec(str1));//null
let str2 = 'cainiao8';
alert(reg.exec(str2));//匹配
//注意：括号中的内容(?=8)只是条件，并不参与真正的捕获，只是检查后面的字符是否符合自己的条件而已，所以，上面的正则返回的是cainiao，而不是cainiao8

/*
*(?!pattern)负向预查
*与正向预查相反，要匹配的字符串，必须不满足pattern这个条件
***************
*/

//下面两个正则表达相同的意思：匹配一个两位数
let reg1 = /^\d{2}$/;
let reg2 = /(?=^)\d{2}(?=$)/;

# 常用正则表达式

1、验证是否为有效数字

/*
*规则分析：
*1、首位可能出现+、-号，也可能不出现。[+-]?
*2、一位数0到9都可以，多位数首位不能是0。(\d|([1-9]\d+))
*3、小数部分可能有可能没有，一旦有，后面必须是小数点加数字。(\.\d+)?
*/
let reg  = /^[+-]?(\d|([1-9]\d+))(\.\d+)?$/;

# 正则的捕获

实现正则捕获的方法

  • 正则RegExp.prototype上的方法
    • exec
    • test
  • String.prototype上支持正则的方法
    • replace
    • match
    • split。。。

捕获的懒惰性

~(function() {
  function execAll(str) {
    let arr = [],
      res = this.exec(str);
    if (!this.global) return res; //如果不是全局匹配，则不执行循环
    while (res) {
      arr.push(res[0]);
      res = this.exec(str);
    }
    return arr;
  }
  RegExp.prototype.execAll = execAll;
})();
let str = "hehe2019haha2020it2048";
let reg3 = /\d+/g;
console.log(reg3.execAll(str)); //["2019", "2020", "2048"]
// 或者 使用字符创的方法-match，execAll为match源码
console.log(str.match(reg3)); //["2019", "2020", "2048"]

分组捕获和分组引用

圆括号()在正则中的两个作用：

  • 1.提高优先级，加括号的优先执行。[+-]===(+|-)，取+-中的一个
  • 2.分组捕获。如果只提高优先级不捕获，则在括号内部加上“?:”，(?:+|-)

分组引用

//分组引用就是通过“/数字”让其代表和对应分组出现一模一样的内容
let str = 'book';	//book,foot,good...
let reg = /^[a-z]([a-z])\1[a-z]$/i;

正则捕获的贪婪性

贪婪性：默认情况下，正则捕获的时候，是按照当前正则所匹配的最长结果来获取的

let str = 'husheng2019znn2020';
let reg = /\d+/g;
str.match(reg);//["2019", "2020"]
//在量词元字符后面设置“?”，取消捕获时的贪婪性（按照正则匹配的最短规则来获取）
reg2 = /\d+?/g;
str.match(reg);//["2", "0", "1", "9", "2", "0", "2", "0"]

正则中问号（?）的5大作用

  • 1、问号左边是非量词元字符：该问号本身就是代表量词元字符，出现0或1次
  • 2、问号左边是量词元字符：取消捕获时的贪婪性
  • 3、(?:)只匹配，不捕获
  • 4、(?=)正向预查
  • 5、(?!)负向预查

replace方法实践

//1、转换日期
let str4 = "2019-12-9";
let reg4 = /^(\d{4})-(\d{1,2})-(\d{1,2})$/;
let res4 = str4.replace(reg4, "$1年$2月$3日");  //2019年12月9日

//2、单词首字母大写;
let str5 = "good good study,day day up!";
let reg5 = /\b([a-zA-Z])[a-zA-Z]*\b/g;
// 函数被执行了6次，每一次都把正则匹配到的信息传递给函数
let res5 = str5.replace(reg5, (...arg) => {
  // console.log(arg);
  let [content, $1] = arg;
  $1 = $1.toUpperCase();
  content = content.substring(1);
  return $1 + content;
});
console.log(res5);	//Good Good Study,Day Day Up!

//3、获取最多出现的字母---排序法
let str6 = "hushenghaaznn";
str6 = str6
  .split("")
  .sort((a, b) => a.localeCompare(b))
  .join(""); //aaeghhhnnnsuz
let reg6 = /([a-zA-Z])\1+/g;
let ary6 = str6.match(reg6); //["aa", "hhh", "nnn"]
ary6.sort((a, b) => b.length - a.length); //降序：["hhh", "nnn", "aa"]
let max6 = ary6[0].length,
  res6 = [ary6[0].substr(0, 1)];
for (let i = 1; i < ary6.length; i++) {
  let item = ary6[i];
  if (item.length < max6) {
    break;
  }
  res6.push(item.substr(0, 1));
}
console.log(`出现最多的字母是${res6}，出现了${max6}次`);

//4、金额转换为千分位
'554444355.334565'.replace(/\d{1,3}(?=(\d{3})+(\.\d*)?$)/g, '$&,')
//"554,444,355.334,565"
