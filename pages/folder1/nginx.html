# 基本命令

查找Nginx：whereis nginx

默认安装目录：/usr/local/nginx

启动：进入/usr/local/nginx/sbin，执行 sudo ./nginx,(root角色进入，不用sudo)

停止：sudo ./nginx -s quit

重启：sudo ./nginx -s reload

检测配置文件以及其所在位置：sudo ./nginx -t

# 配置文件分开

在根配置文件中（nginx.conf）的http下添加：include /etc/nginx/conf.d/*.conf;

cd /etc/nginx；cd conf.d；创建文件（域名.conf）,在里面编辑server。

# 3、Nginx的proxy buffer参数总结 (opens new window)

nginx服务器中有较大文件时，比如gltf文件为33.4M时配置如下：

	proxy_buffering on;
	proxy_buffers 8 20M;
	proxy_buffer_size 4k;
	proxy_busy_buffers_size 40M;
