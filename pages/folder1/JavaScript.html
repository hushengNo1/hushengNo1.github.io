使用 typeof 对这些数据类型的值进行操作的时候，返回的结果却不是一一对应，分别是：

console.log(typeof a); // function

var error = new Error();

    // 在 IE6 中，null 和 undefined 会被 Object.prototype.toString 识别成 [object Object],故作兼容

        return obj + "";
    }
    return typeof obj === "object" || typeof obj === "function" ?

        typeof obj;
}

# 防抖节流

都是为了减少频繁的事件触发，比如：

  • window 的 resize、scroll
  • mousedown、mousemove
  • keyup、keydown
  1. 防抖(防反跳)
    1. 参考：JavaScript专题之跟着underscore学防抖 (opens new window)
    2. 原理：你尽管触发事件，但是我一定在事件触发 n 秒后才执行，如果你在一个事件触发的 n 秒内又触发了这个事件，那我就以新的事件的时间为准，n 秒后才执行，总之，就是要等你触发完事件 n 秒内不再触发事件，我才执行。
    3. 应用：
      • 频繁的事件回调中做复杂计算
      • 浏览器窗口拖动变化
      • 搜索引擎输入框，希望用户输入完最后一个字才调用查询接口。
  2. 节流
    1. 参考：JavaScript专题之跟着 underscore 学节流 (opens new window)
    2. 原理：如果你持续触发事件，每隔一段时间，只执行一次事件。
    3. 应用：
      • 滚动时显示、隐藏顶部导航

# 变量提升

变量提升机制：把所有带有var，function关键字的进行提前声明，

  • var：提前声明为undefined，
  • function：声明、定义、赋值全部完成

带var和不带var的区别

a=12;//不存在变量提升，相当于window.a = 12;不加var的本质是window的属性
var a = 12;//声明一个全局变量的同时在window对象下映射一个a的属性并赋值为12（window.a = 12）
console.log(a);//先检查是否存在全局变量a，不存在则然后检查window.a是否存在

var a=b=12;=>var a=12;b=12;

# script标签中的defer和async

  1. defer,async只对外联script脚本文件有效, 内联script脚本设置无效。
  2. 普通script的 下载和执行 都阻塞页面的解析渲染。
  3. defer
  4. 文档解析时，遇到设置了defer的script脚本，就会在后台进行下载，下载并不会阻止文档的解析渲染。
  5. 当页面解析渲染完毕后, 会等到所有的defer脚本下载完毕并按照顺序执行，执行完毕后会触发DOMContentLoaded事件。
  6. async
  7. 文档解析时，遇到设置了async的script脚本，就会在后台进行下载，下载并不会阻止文档的解析渲染。
  8. async脚本的 下载 不会阻塞页面的解析渲染。async脚本的 执行 会阻塞页面的解析渲染。
  9. 哪个async脚本先下载完, 就立刻执行, 执行时阻止文档的解析渲染。

    2. 深对比：递归实现

      function deepEqual(object1, object2) {
  const keys1 = Object.keys(object1);
  const keys2 = Object.keys(object2);

  if (keys1.length !== keys2.length) {
    return false;
  }

  for (const key of keys1) {
    const val1 = object1[key];
    const val2 = object2[key];
    const areObjects = isObject(val1) && isObject(val2);
    if (
      (areObjects && !deepEqual(val1, val2)) ||
      (!areObjects && val1 !== val2)
    ) {
      return false;
    }
  }

  return true;
}

function isObject(object) {
  return object != null && typeof object === "object";
}
