# 基本命令

git config --global --list 查看本地git配置

git config --global user.name 你的用户名 //设置全局用户名

git config --global user.email 你的登陆邮箱 //设置邮箱

git remote show origin”查看当前仓库基本信息

git remote -v 查看当前仓库对应的远程仓库地址

git remote set-url origin 仓库地址 //修改仓库对应的远程仓库地址

git add -A或git add . 可以提交当前仓库的所有改动

git status -s 查看仓库当前文件提交状态（A：提交成功；AM：文件在添加到缓存之后又有改动）

git commit -m "1.0.0" 从Git的暂存区提交版本到仓库，参数-m后为当次提交的备注信息

git commit -a -m "massage" -a参数可以将所有已跟踪文件中的执行修改或删除操作的文件都提交到本地仓库，即使它们没有经过git add添加到暂存区，注意，新加的文件（即没有被git系统管理的文件）是不能被提交到本地仓库的。

git log：查看提交记录

git log --oneline：输出概要日志

git log --oneline -5：指定最近几个提交可以带上 - + 数字

git reflog：查看所有的历史记录（包括历史区回滚信息）

git reset --hard 6fdfbc47f13df7d2b580a85a6e1fad5e10325d7d：回归到某一个版本

git push https://gitee.com/***/test.git 将本地的Git仓库信息推送上传到指定服务器仓库
