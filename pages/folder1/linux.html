# !vim编辑

按ESC键跳到命令模式，然后：

  • :w - 保存文件，不退出 vim。
  • :w file -将修改另外保存到 file 中，不退出 vim。
  • :w! -强制保存，不退出 vim 。
  • :wq -保存文件，退出 vim。
  • :wq! -强制保存文件，退出 vim。
  • :q -不保存文件，退出 vim。
  • :q! -不保存文件，强制退出 vim。
  • :e! -放弃所有修改，从上次保存文件开始再编辑。
  • 打开行号 进入编辑之后:输入:set nu

# 查看文件

# 基本命令（一）：

  1. nc -zvw3 8.136.83.212 7443 检查7443端口是否开通

    • nc：即执行的命令主体；
    • z：零 I/O 模式（被用来扫描）；
    • v：显式地输出；
    • w3：设置超时时间为 3 秒；

# 基本命令（二）：

1、netstat -tunlp查看端口使用情况；mac：使用 lsof -i :7001来查看端口号为7001的pid

2、kill -9 [pid] 杀死进程

3、查看当前操作系统版本信息 cat /proc/version

4、全局搜索文件：find / -name nginx.conf

5、删除指定后缀文件：find . -name '*.png' -type f -print -exec rm -rf {} ;

6、获取ip：ifconfig en0

7、删除文件：rm -f [文件];删除文件夹：rm -rf [文件夹]，新建文件：touch Test.sh

8、uname可显示电脑以及操作系统的相关信息。参 数： -a或--all 显示全部的信息。 -m或--machine 显示电脑类型。 -n或-nodename 显示在网络上的主机名称。 -r或--release 显示操作系统的发行编号。 -s或--sysname 显示操作系统名称。 -v 显示操作系统的版本。

9、cat ~/.bash_history查看输入的命令历史。一般在root目录下面

10、ls /etc | grep 'ssh' 过滤出 /etc 目录中名字包含 ssh 的目录(不包括子目录)；Linux 中管道的作用是将上一个命令的输出作为下一个命令的输入, 像 pipe 一样将各个命令串联起来执行, 管道的操作符是 |。

11、echo 'Hello World' > ~/test.txt；可以使用 > 或 < 将命令的输出重定向到一个文件中

12、ping -c 4 cloud.tencent.com；对 cloud.tencent.com 发送 4 个 ping 包, 检查与其是否联通

13、cp -r /home/install-package/* /home/cp/install_package/

将/home/install-package/文件夹内的所有文件都复制到 /home/cp/install_package/文件夹下（可用相对路径）。

cp ./objs/nginx /usr/local/nginx/sbin/ 将objs文件夹下的/nginx文件复制到/usr/local/nginx/sbin/文件夹下

14、打印环境变量 echo $PATH，必须大写path。linux添加环境变量参考https://www.cnblogs.com/joshua317/p/6899057.html

15、压缩当前文件夹下非隐藏文件：tar czvf test.tar.gz *，解压：tar -xvzf test.tar.gz ;

​ 将tgz文件解压到指定目录tar zxvf test.tgz -C 指定目录

	- 压缩：tar -zcvf ./node_modules.tar.gz ./node_modules
	- 解压：tar -zxvf ./node_modules.tar.gz

16、查看当前登陆用户：who

17、显示当前与过去登录系统的用户的信息：last

18、显示所有登录系统失败的用户的信息：lastb

19、curl http://localhost:8080

20、想操作某个目录，或者某个目录下的文件，提示权限不够； 最简单的方法：chmod 777 文件/文件夹名称 赋给所有用户这个文件/文件夹的读，写，看权限；

  1. 查看文件系统类型:df -hT
  2. ping的一些用法 (opens new window)
