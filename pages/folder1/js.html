# 函数

变量提升机制：把所有带有var，function关键字的进行提前声明，

  • var：提前声明为undefined，
  • function：声明、定义、赋值全部完成

带var和不带var的区别

a=12;//不存在变量提升，相当于window.a = 12;不加var的本质是window的属性
var a = 12;//声明一个全局变量的同时在window对象下映射一个a的属性并赋值为12（window.a = 12）
console.log(a);//先检查是否存在全局变量a，不存在则然后检查window.a是否存在

var a=b=12;=>var a=12;b=12;

在ES3/ES5语法规范中，只有全局作用域和函数执行的私有作用域（栈内存），其他大括号不会形成栈内存

变量提升阶段，函数不管写在哪个位置，在对应的作用域中都优先完成声明和定义；当函数执行时之前，开辟当前函数私有作用域（私有的栈内存），函数私有作用域中第一件事同样也是变量提升（变量提升前，先形参赋值）；

  • 函数私有作用域形成之后：形参赋值&&变量提升&&函数内代码执行

# 知识点

2、文档卷起的高度兼容写法：

// 滚动条在Y轴上的滚动距离
document.documentElement.scrollTop：Chrome
window.pageYOffset：Chrome、IE
document.body.scrollTop：IE
const scrollH = document.documentElement.scrollTop || window.pageYOffset || document.body.scrollTop;
//或者
const scrollTop = Math.max(
  window.pageYOffset,
  document.documentElement.scrollTop,
  document.body.scrollTop
)
//再或者
function getBodyScrollTop() {
  const el =
    document.documentElement || document.scrollingElement || document.body
  return el.scrollTop
}
